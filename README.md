0xZAXOYI — Sensitive File Scanner Advanced Chrome Extension for Ethical
Web Security Testing

⚠️ Ethical Use This extension is intended only for authorized security
testing: - Applications you own - Systems you have written permission to
test - Bug bounty programs within scope

Unauthorized scanning is illegal.

Overview 0xZAXOYI Sensitive File Scanner detects exposed sensitive files
during legitimate assessments. Non-invasive, safe HTTP reconnaissance.

Features - Scans 800+ sensitive paths (backups, configs, logs, VCS
exposures, CMS files, admin panels, cloud leaks) - Real-time status:
Critical / Warning / Safe - Auto-detected domain - JSON export - Local
scan history - Category toggles - Manifest V3, concurrency control, soft
404 detection

Installation git clone https://github.com/MohamedKrd/Sensitive-Files-Scanner.git cd
0xZAXOYI Load unpacked in chrome://extensions/ after enabling Developer
Mode.

Usage 1. Navigate to authorized target 2. Open extension 3. Confirm
domain 4. Start scan 5. Review results

Structure 0xZAXOYI/ - manifest.json - pages/ - js/ - css/ -
data/payloads.json - icons/ - README.md

Security & Privacy Local-only storage, no telemetry, passive GET
scanning.
